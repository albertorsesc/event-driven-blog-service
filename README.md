## Event Driven microservice structure for a Blog

#### Start Services

**Client:**

* cd client;
* npm install;
* npm run start;

**Comments:**

* cd comments;
* npm install;
* npm run start;

**Comments Moderator:**

* cd moderation;
* npm install;
* npm run start;

**Posts:**

* cd posts;
* npm install;
* npm run start;

**Event Bus:**

* cd event-bus;
* npm install;
* npm run start;
